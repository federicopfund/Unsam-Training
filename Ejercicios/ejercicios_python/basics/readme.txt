This folder contains the Python files for the exercises of the first class.

Files:
- shape_calculations.py: Basic shape classes and area calculations.
- geringoso_converter.py: Converter for Geringoso language.
- mortgage_calculator.py: Mortgage calculation exercises.
- inclusive_range.py: Inclusive range implementation.
- bounces.py: Bouncing ball simulation.
- greeting.py: Simple greeting program.
